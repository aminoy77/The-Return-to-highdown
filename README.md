🐍 The Legends of Highdown

Un pequeño proyecto desarrollado con Pygame Zero.

Requisitos

- Python 3.8 o superior
- Pygame Zero (pgzero)

Instalación y ejecución

Opción 1: Ejecución rápida (recomendada)

1. Descarga o clona este repositorio
2. Coloca todos los archivos en la misma carpeta, incluyendo la carpeta images/
3. Haz doble clic en jugar.bat (Windows) o jugar.sh (Linux/macOS)
   - La primera vez instalará pgzero automáticamente si no lo tienes
   - Después simplemente iniciará el proyecto

Opción 2: Ejecución manual desde terminal

pip install pgzero
python -m pgzero vida.py
# o en algunos sistemas:
pip3 install pgzero
python3 -m pgzero vida.py

Nota: asegúrate de ejecutar los comandos desde la carpeta del proyecto.
